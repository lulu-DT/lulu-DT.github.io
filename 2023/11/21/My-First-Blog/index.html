Introduce

Explain how to deploy a personal blog on a cloud server

Background

  • you need a cloud server
  • ubuntu 22.04
  • git

Step

1. install Node.js

  • download the node.js package
  • decompress package
  • add the executable path to env ${PATH}
  • check the node version
  • config npm permissions

If you do not configure this, you will encounter permission denied later

2. install Hexo

  • config source to accelerate download
